This project is moved to: https://mhogomchungu.github.io/sirikali

Reason?

I initially named the project "cryfs-gui" because it initially acted as a front end to only "cryfs" tool but this is no longer the case since it now acts as a front end to "cryfs","gocryptfs","securefs" and "encfs" and i felt the project needed a neutral name and i settled with "SiriKali".
